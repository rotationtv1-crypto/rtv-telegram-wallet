# RotationTV Network — Unified Architecture Blueprint
One Cloudflare Worker. One Supabase. One Vercel API. 9 companies. 1 Telegram Mini App.

## Stack
- Cloudflare Workers: DOs, Queues, R2, Analytics, Browser Rendering, Workflows
- Vercel: AI Gateway, tRPC, OAuth, Image gen
- Supabase: Realtime, PostgreSQL, RLS, Storage, Auth
- TON/Solana: $RTVS/$RTV, Chainstack nodes
- Base44: 30+ entities, backend functions

## Deploy
1. npx wrangler deploy
2. vercel --prod  
3. Set Telegram Mini App URL
